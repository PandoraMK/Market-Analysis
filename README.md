Gold & Silver Market Analysis: A Multi-Factor Approach

Overview
This project analyses the relationship between gold (XAU) and silver (XAG) returns and key macroeconomic and financial market variables. 
The goal is to understand whether factors such as inflation, monetary policy, and market sentiment explain commodity price movements.

Objectives
- Analyse the relationship between gold/silver returns and macroeconomic indicators
- Explore the impact of:
  - CPI (inflation)
  - NFP (employment)
  - FOMC (monetary policy)
  - DXY (US Dollar Index)
  - S&P 500 (market sentiment)
  - USOIL (economic activity)
- Build and evaluate a regression model
- Interpret financial and economic implications

Tools & Technologies
- Python (Pandas, NumPy)
- Matplotlib & Seaborn (visualisation)
- Statsmodels (OLS regression)
- Jupyter Notebook

Key Insights
- Weak linear relationships were found between gold returns and macroeconomic variables
- Market behaviour appears to be influenced by:
  - Expectations rather than realised data
  - Non-linear dynamics
  - Global sentiment and risk factors
- Volatility clustering observed in gold returns

Limitations
- Differences in data frequency (daily vs monthly)
- No lagged variables included
- Linear model may not capture complex relationships

Future Improvements
- Add lagged variables (VERY IMPORTANT)
- Explore non-linear models / machine learning
- Include interest rates and geopolitical risk indicators
